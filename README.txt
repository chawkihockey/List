List.java: Class defines ADT used to store the
sorted indices of an array that contains the words
of an input text file. Exports its services to
client class Lex.java

ListClient.java: A class that tests the class List and its
associated methods.

Lex.java: Executes the requirements for the assignment.
Uses List class to create a List, reads in the lines
of a file to an array, then sorts the indices into
a List object, then uses the List to output the 
correct order into an output file.

Makefile: Creates executable jar file called Lex
